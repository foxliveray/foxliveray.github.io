素因子分解

发表于 2018-01-22

时间感觉越来越紧迫了，自己不会的东西很多，保研心里没底。所以从今天开始，每天都刷一些数据结构和算法的题吧，poj，zoj，PAT，北大百炼，leetcode，code wars，newcoder这些。博客就用来记录一下刷题过程中的一些知识点、感悟和总结吧。

今天完成的第一题是2017北大计算机研究生推免上机考试的第一题：素因子分解

描述

输入一个数，输出其素因子分解表达式。

输入

输入一个整数 n (2 <= n < 100)。

输出

输出该整数的因子分解表达式。
表达式中各个素数从小到大排列。
如果该整数可以分解出因子a的b次方，当b大于1时，写做 a^b ；当b等于1时，则直接写成a。

分析

其实一开始自己分析的时候，对一个整数n，觉得要从最小的质数2开始，一直往上找，找到n-1为止，去判断n是否能被这些因数中为质数的因数整除。那么，这个问题的难点就在于，怎么去在0到n的范围内，哪些数字为质数。要找出所有在范围内的素数，其实有一个方法，“埃拉托斯特尼筛法”，要得到自然数n以内的全部素数，必须把不大于 的所有素数的倍数剔除，剩下的就是素数。
给出要筛数值的范围n，找出以内的素数。先用2去筛，即把2留下，把2的倍数剔除掉；再用下一个质数，也就是3筛，把3留下，把3的倍数剔除掉；接下去用下一个质数5筛，把5留下，把5的倍数剔除掉；不断重复下去……。

不过，后来发现其实不用这么做，因为当你用n去除2时，因为会循环地去除2，所以只要是2的倍数的都不用去考虑了，首先他们也不是质数，其次当用n去除4,6,8,10······时，因为在除2时，已经除去了，所以这些都已经不是n的因数了。

另外，这是整数的素因子分解问题，一个整数n的素数因子不可能同时大于sqrt(n）的，所以循环的上限就是sqrt(n)。逐一查看这些素数是否是n的素因子即可，要注意最后除完所有素因子后如果结果>1,这个数也是素因子，所以到这里基本上解题的思路就差不多了，剩下的就是不断地调试，修改判断条件等，直到符合样例输出即可。
